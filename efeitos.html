Dupla Exposição

A sobreposição de duas imagens para criar uma única composição artística. Esse efeito pode ser usado de várias maneiras para criar resultados visualmente interessantes.

Efeito Bokeh

Destaque de pontos de luz desfocados no fundo da imagem para criar um efeito bokeh. Isso é comumente usado em retratos para adicionar um toque de glamour.
